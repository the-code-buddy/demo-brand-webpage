# Demo Brand Webpage 🧥

A clean, minimalist, and modern landing page designed for an apparel and clothing brand, specifically showcasing a sweater collection. 

🌐 **Live Demo:** [View Website Here](https://the-code-buddy.github.io/demo-brand-webpage/)

## 📖 About The Project
This project is a static e-commerce front-end demonstration. It features a sleek user interface aimed at highlighting fashion products with high-quality imagery and a simple, readable layout.

### ✨ Key Features
* **Modern Navigation:** Includes links for Home, Shop, Lookbook, About, News, and Cart.
* **Product Showcase:** A dedicated section displaying product details and material information (e.g., 100% Cotton).
* **Lookbook / Gallery:** A "New Collection" section featuring high-quality placeholder images of models wearing sweaters (sourced from Unsplash).
* **Minimalist UI:** Focuses on typography, spacing, and large imagery for a premium brand feel.

## 🛠️ Built With
* **HTML5:** For the semantic structure of the webpage.
* **CSS3:** For styling, layout, and responsive design.
